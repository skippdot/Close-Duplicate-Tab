Overview

Close Duplicate Tab is a Chrome extension that helps users manage browser tab clutter by automatically identifying and closing duplicate tabs with identical URLs. This privacy policy explains how the extension handles data and protects your privacy.

Key Features: Automatic duplicate detection, smart tab preservation, tab suspender compatibility, real-time statistics, search functionality, and professional dashboard interface.

Data Collection

üö´ Zero Data Collection: We do not collect any personal data, browsing history, or personally identifiable information. The extension operates entirely within your browser and does not transmit any information to external servers.

Data Access & Usage

The extension accesses the following data solely for its core functionality:

  • Tab URLs: Read to identify duplicate tabs with identical URLs, including extraction of real URLs from suspended tabs (Marvellous Suspender compatibility)
  • Tab Titles: Read to display in the statistics dashboard and domain listings
  • Tab States: Read to determine which tabs are active, pinned, suspended, or in different browser windows
  • Window Information: Read to support "Current Window Only" functionality

üîÑ Local Processing Only: All data is processed locally in your browser in real-time. No data is stored permanently, cached, or transmitted to external servers. When you close the extension, all processed data is immediately discarded.

Data Storage

The extension stores only user preferences using Chrome's secure sync storage:

  • autoClose: Your preference for automatic duplicate closing (boolean: true/false)
  • currentWindowOnly: Your preference to limit duplicate detection to the current window only (boolean: true/false)
  • sortTabs: Your preference to reorganize all tabs alphabetically by URL when closing duplicates (boolean: true/false)

üìù Settings Storage: These preferences are stored locally on your device and synced across your Chrome browsers if you're signed into Chrome. This sync is handled entirely by Google Chrome's built-in sync service - we do not have access to this data.
