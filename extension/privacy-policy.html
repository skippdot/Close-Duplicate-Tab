Privacy Policy for Close Duplicate Tab

Last updated: July 25, 2025

Overview

Close Duplicate Tab is a Chrome extension that helps users identify and close duplicate browser tabs with identical URLs. This privacy policy explains how the extension handles data.

Data Collection

We do not collect any personal data. The extension operates entirely within your browser and does not transmit any information to external servers.

Data Usage

The extension accesses the following data solely for its functionality:

  • Tab URLs and Titles: Used to identify duplicate tabs with identical URLs
  • Tab States: Used to determine which tabs are active, pinned, or in different windows

This data is processed locally in your browser and is never stored permanently or transmitted externally.

Data Storage

The extension stores only user preferences using Chrome's sync storage:

  • sortTabs: Whether to reorganize tabs by URL when closing duplicates
  • autoClose: Preference for automatic duplicate closing (feature not yet implemented)
  • currentWindowOnly: Preference to limit duplicate detection to current window (feature not yet implemented)

These preferences are stored locally and synced across your Chrome browsers if you're signed into Chrome.

Permissions

The extension requires the following permissions:

  • tabs: Required to read tab URLs/titles, identify duplicates, and close duplicate tabs
  • storage: Required to save your preferences

Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

Data Security

All data processing occurs locally within your browser. No data is transmitted to external servers or third parties.

Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date at the top of this page.

Contact

If you have questions about this privacy policy, please contact us through the Chrome Web Store listing.
